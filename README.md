# Exercícios de Revisão - Map e Desestruturação (Preparação para React)

Este repositório contém uma lista de exercícios em JavaScript com foco em revisão de conceitos essenciais para o desenvolvimento com React.

---

## Objetivo

Reforçar os principais fundamentos de JavaScript moderno que são amplamente utilizados no React, como:

- Manipulação de arrays com `map()`
- Desestruturação de objetos e arrays
- Renomeação de variáveis na desestruturação
- Uso de parâmetros desestruturados em funções
- Spread operator (`...`)
- Transformação de dados em arrays de objetos
- Lógica funcional aplicada a dados

---

## Por que isso é importante para React

Esses conceitos são usados constantemente no React, especialmente em:

- Renderização de listas com `.map()`
- Props com desestruturação
- Manipulação de estados (state)
- Transformação de dados vindos de APIs
- Componentização de dados
- Imutabilidade com spread operator

---

## Estrutura dos Exercícios

Os exercícios podem estar organizados em um único arquivo ou separados:


exercicios.js


---

## Conteúdo dos Exercícios

### 1. Renomeação de variáveis
Uso de desestruturação com renomeação de propriedades de objetos.

### 2. Formatação de dados
Transformação de arrays de objetos em strings formatadas.

### 3. Desestruturação em parâmetros
Extração de propriedades diretamente na assinatura de funções.

### 4. Map com objetos
Criação de novos objetos a partir de arrays de números.

### 5. Desestruturação aninhada
Acesso direto a propriedades dentro de objetos complexos.

### 6. Lógica condicional no map
Classificação de valores dentro de arrays.

### 7. Desestruturação de arrays aninhados
Extração de valores de estruturas complexas.

### 8. Cálculo com atualização de objetos
Aplicação de desconto usando spread operator.

### 9. Extração de dados de API simulada
Redução de objetos complexos para estruturas simples.

### 10. Uso do array completo no map
Cálculo de proporções com base no array original.
